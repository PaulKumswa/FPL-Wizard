How It Works

▼
📊

Data Pipeline

Source: Official FPL API + Understat (xG/xA)

Process: 5-week rolling averages for Form, xG, xA, Team xGA

Update: Weekly (before gameweek)

🧠

Machine Learning

Approach: Component-based prediction (Goals, Assists, Clean Sheets)

Models: ~10 LightGBM classifiers (per position × per outcome)

Output: P(goal) × pts + P(assist) × 3 + P(cs) × pts + 2

🎯

Inference Engine

Selection: Top 1 per position + 1 Wildcard

Confidence First: Prioritizes high-confidence (≥60%) 6+ pt picks before relaxing criteria

Fail-Upward: Relaxes confidence → ownership → cost → points threshold
